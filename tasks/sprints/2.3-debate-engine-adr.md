## Design (keep simple, immutable, English)
1. `@dataclass(frozen=True) Position`: `role: str`, `stance: str` (one-line),
   `rationale: str`.
2. `@dataclass(frozen=True) DebateResult`: `topic: str`, `positions: tuple[Position, ...]`,
   `decision: str`, `consequences: str`.
3. `class DebateService`:
   - `__init__(self, vault_path: Path | str | None, dry_run: bool = True)`.
   - `synthesize(self, topic: str, positions: list[Position], decision: str | None = None,
     consequences: str | None = None) -> DebateResult` â€” pure function. If `decision` is None,
     apply a SIMPLE deterministic synthesis rule (e.g. choose the position that the most roles
     share by `stance`, ties â†’ first by input order) and record it; do NOT call any LLM.
     If `consequences` is None, derive a short summary listing dissenting positions.
   - `to_adr(self, result: DebateResult, *, security_impact: str = "None identified",
     review_date: str = "") -> dict | None` â€” calls `write_adr(title=topic,
     context=<topic + the debate framing>, options=[f"{p.role}: {p.stance}" for p in positions],
     decision=result.decision, consequences=result.consequences, security_impact=...,
     review_date=...)`. Returns the emit dict, or None if no vault.
   - `run(self, topic, positions, decision=None, consequences=None, security_impact=...,
     review_date=...) -> dict | None` â€” convenience: synthesize then to_adr.
4. dry_run default TRUE; no real write unless dry_run=False AND vault exists.

## Tests (`tests/test_debate_service.py`) â€” tmp vault
- `synthesize` with explicit decision returns that decision in `DebateResult`.
- `synthesize` with decision=None applies the majority-stance rule deterministically
  (construct positions where 2/3 share a stance â†’ that stance wins).
- `to_adr` passes `options` as a `list[str]` (one per position) â€” assert the written content
  includes each role's stance.
- `run` with `vault_path=None` returns None (no-op).
- dry_run=True returns dict with `dry_run is True` and writes NO file; dry_run=False + tmp vault
  writes a file under `12 - HivePilot/03 - Decisions/`.
