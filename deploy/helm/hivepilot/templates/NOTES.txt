{{- if not .Values.secrets.apiTokensYaml }}

WARNING: secrets.apiTokensYaml is empty — the API started with an EMPTY
token store. Every authenticated /v1 endpoint returns 401 until you
bootstrap a real admin token. See deploy/helm/hivepilot/README.md
"Bootstrap the first admin token" for the exact steps, then re-run
`helm upgrade` with the generated api_tokens.yaml content (or
`--set-file secrets.apiTokensYaml=./api_tokens.yaml`).
{{- end }}

{{- if and .Values.scheduler.enabled (not (hasKey .Values.secrets.data "HIVEPILOT_API_TOKEN")) }}

WARNING: scheduler.enabled=true but secrets.data.HIVEPILOT_API_TOKEN is not
set — the scheduler daemon will fail its `_require_cli_role` check on
startup and refuse to run schedules. Set
secrets.data.HIVEPILOT_API_TOKEN to a bootstrapped token with role >= "run".
{{- end }}
