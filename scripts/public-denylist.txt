# Public-safe denylist
#
# Structural (non-sensitive) regex patterns that indicate un-genericized
# internal/org-specific content leaking into this PUBLIC repo's shipped
# example/default prompts. One pattern per line, `#` starts a comment,
# blank lines are ignored. Patterns are compiled case-insensitively.
#
# IMPORTANT: do NOT add real personal names here — this file is committed
# to a public repo, and enumerating names would re-leak them. Personal-name
# patterns (or any other sensitive-but-structural pattern maintainers don't
# want in a public file) belong in a PRIVATE, uncommitted file referenced
# via the PUBLIC_DENYLIST_EXTRA environment variable, e.g.:
#
#   PUBLIC_DENYLIST_EXTRA=~/.hivepilot-private/denylist.txt \
#     python scripts/check_public_safe.py
#
# That file is never committed to this repo.
